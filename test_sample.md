# MDViewer Windows TTS Test

This is a test file to verify that the Windows text-to-speech functionality works correctly.

## Test Sections

### Section 1: Basic Text
The quick brown fox jumps over the lazy dog. This sentence contains all letters of the alphabet.

### Section 2: Technical Content
MDViewer now supports Windows using the Speech API also known as SAPI. The integration uses PowerShell to access the System.Speech.Synthesis namespace.

### Section 3: Numbers and Punctuation
Testing numbers: 1, 2, 3, 100, 250. Testing punctuation! Does it work? Yes, it does.

## Conclusion
If you can hear this being read aloud on Windows, the TTS implementation is working perfectly!
